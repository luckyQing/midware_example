# kafka安装
## 一、安装zookeeper
- 下载
  ```
  https://zookeeper.apache.org/releases.html#download
  ```
- 启动
  ```
  双击bin/zkServer.cmd
  ```

## 二、安装kafka
- 下载
  ```
  https://kafka.apache.org/downloads
  ```
- 启动
  ```
  C:/dev/kafka_2.13-3.5.1/bin/windows/kafka-server-start.bat  C:/dev/kafka_2.13-3.5.1/config/server.properties
  ```
